| | 7:30-8:30 | **ハンズオン：** ナレッジベースの作成。CSVやテキストファイルをアップロードし、AIが参照できるデータソースを準備する [5]。 | RAG実装の第一歩として、データ連携の方法を学ぶ。 |
| | 8:30-10:00 | **ハンズオン：** 作成したチャットボットにナレッジを連携させる。コンテキストとしてナレッジを追加し、RAGの基本を実装する [5]。 | ナレッジに基づいた回答ができるボットを作成し、RAGの効果を実感する。 |

| | 10:30-11:30 | **ハンズオン：** Notionでナレッジベースを構築。提供されたサンプル規定をNotionに整理し、Difyと連携させる [7]。 | ビジネス現場で多用されるツールとの連携スキルを習得する。 |

### 3. 『生成AI×SNS運用講座』詳細タイムライン（全15時間）

| モジュール | 時間 | 学習内容 | 目的・ポイント |
| :--- | :--- | :--- | :--- |
| **Module 1: AI時代のSNSマーケティング戦略** | **2時間** | | |
| | 0:00-0:30 | **オリエンテーション：** AIがSNS運用（戦略、制作、分析）をどう変えるか。PDCAサイクル高速化の重要性 [19, 20]。 | 勘と経験頼りの運用から脱却するためのマインドセットを構築する。 |

| | 1:15-2:00 | **ハンズオン：** AIで競合アカウント分析。競合の強みや投稿傾向を分析させ、自社のポジショニングを検討する [22]。 | 市場を理解し、差別化されたコンテンツ戦略を立案する。 |
