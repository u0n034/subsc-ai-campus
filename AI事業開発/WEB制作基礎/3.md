#### 1.1 コピーライティングとは何か：言葉で「行動」を生み出す技術（10分）

**コピーライティングの定義**
- コピーライティング = 「人を動かす文章術」
- 情報を「伝える」のではなく、読み手に「行動してもらう」ことが目的
- LPにおけるコピーの役割：訪問者を「問い合わせ」「資料請求」「購入」へ導く

**コピーライティングの3つの機能**
1. **Attention（注意）：** 見出しで注意を引く
2. **Engagement（関与）：** 本文で興味を持続させる
3. **Action（行動）：** CTAで具体的な行動を促す

**良いコピーの4つの条件**
1. **具体性：** 抽象的ではなく、数字・固有名詞・情景描写を使う
2. **共感性：** ターゲットの「悩み」や「願望」に寄り添う
3. **独自性：** 競合にはない「自社だけの価値」を明確にする
4. **行動性：** 読み手に「次に何をすべきか」を明示する

#### 1.2 Webマーケティング基本用語：LPの解剖学（15分）

**LP（ランディングページ）とは**
- 訪問者が最初に「着地（Landing）」するページ
- 通常のWebサイトとの違い：1つの目的（コンバージョン）に特化
- LPの構造：縦長の1ページ完結型が主流

**LPの主要構成要素と用語解説**

1. **ファーストビュー（First View / Above the Fold）**
   - 定義：ページを開いた瞬間、スクロールせずに見える範囲
   - 重要性：訪問者の離脱判断は3秒以内に起こる
   - 配置すべき要素：メインキャッチコピー、サブヘッドライン、メインビジュアル、第1CTA

2. **見出し（Headline / H2, H3）**
   - 定義：各セクションの冒頭に配置される短いテキスト
   - 役割：訪問者を次のブロックへ誘導する「道しるべ」
   - 種類：
     - キャッチコピー（最上部の最も目立つ見出し）
     - セクション見出し（各ブロックの区切り）

3. **ボディコピー（Body Copy）**
   - 定義：見出しの下に配置される説明文・本文
   - 役割：訪問者に「理解」と「納得」を与える
   - 推奨文字数：スマホで読みやすい200〜400文字/セクション

4. **CTA（Call To Action：行動喚起）**
   - 定義：訪問者に具体的な行動を促すボタンや文言
   - 配置：複数箇所（ファーストビュー、中間、最下部）
   - 良いCTAの条件：
     - 行動が明確（「お問い合わせ」より「無料相談を予約する」）
     - ハードルが低い（「購入」より「資料請求」）
     - ベネフィットが見える（「詳しく見る」より「あなたに合ったプランを見る」）

5. **マイクロコピー（Microcopy）**
   - 定義：CTAボタン周辺の小さな補足説明文
   - 役割：訪問者の「不安」を解消し、行動のハードルを下げる
   - 例：「30秒で完了」「無料・営業電話なし」「後からキャンセル可能」

6. **コンバージョン（Conversion / CV）**
   - 定義：訪問者が起こした「成果となる行動」
   - 種類：
     - 直接CV：購入、契約、申し込み
     - 間接CV：問い合わせ、資料請求、会員登録
   - 測定指標：CVR（コンバージョン率）= CV数 ÷ 訪問者数

**その他の重要用語**
- **レスポンシブデザイン：** PC・タブレット・スマホで最適表示を切り替える設計
- **スクロール率：** 訪問者がページのどこまで読んだかの割合
- **離脱率：** そのページで閲覧を終了した訪問者の割合
- **ヒートマップ：** 訪問者の視線やクリックを可視化したツール

#### 1.3 本章のゴールと学習ロードマップ（5分）

**第3章で作成する成果物**
1. ファーストビューのメインキャッチコピー（1案に絞る）
2. ファーストビューのサブヘッドライン
3. 各セクションの見出し（H2レベル）10〜12個
4. 重要セクションのボディコピー（3セクション分）
5. CTAボタン文言とマイクロコピー

**学習の流れ（3時間）**
- セクション1（現在）：基礎理解
- セクション2（60分）：キャッチコピーと見出しを作る
- セクション3（60分）：ボディコピーとCTAを作る
- セクション4（30分）：全体を編集・ブラッシュアップする

**AIとの協業スタイル**
- AIの役割：「たたき台」を高速で大量生成
- 人間の役割：批判的に検証し、ペルソナに最適な案を選択・編集
- 目指すゴール：坂本さん（ペルソナ）の心を動かし、「問い合わせ」という行動を起こさせるコピー

### セクション2：【実践1】キャッチコピーとヘッドラインの作成
**時間：** 4:30:00 - 5:30:00（60分）

**目的：**
- LP各セクションの「見出し」を作成する
- AIに「制約」を与えて精度を高める技術を習得する
- 「良いコピー」と「悪いコピー」を見分ける基準を学ぶ

**内容構成：**

#### 2.1 コンセプト：キャッチコピーの「3つの型」（10分）
- **共感型：** ペルソナの「痛み」に寄り添う（例：「何から始めればいいか分からない」その迷い、私たちが解決します）
- **未来提示型：** ペルソナの「願望」を可視化する（例：あなたは22時の残業をやめて、家で夕食をとれます）
- **独自性主張型：** 競合との「差別化」を明示する（例：ツールは選びません。'あるもので回す'設計で始めましょう）

#### 2.2 ワークショップ（実践）：ファーストビューのコピーを作る（50分）

**【ステップ1：メインキャッチの生成】（15分）**

**【ステップ2：サブヘッドラインの作成】（15分）**

**【ステップ3：各セクション見出しの一括生成】（20分）**

### セクション3：【実践2】ボディコピーとCTAの作成
**時間：** 5:30:00 - 6:30:00（60分）

**目的：**
- 各セクションの「本文（説明文）」を作成する
- CTAボタンの文言を最適化する
- トーン＆マナー（文体・語調）の統一を学ぶ

**内容構成：**

#### 3.1 コンセプト：ボディコピーの「3つの黄金ルール」（10分）
1. **具体性：** 抽象的な言葉を避け、数字・固有名詞・情景描写を使う
2. **一貫性：** ペルソナの「言葉」で語る（専門用語ではなく実務用語）
3. **リズム：** 長文を避け、短い文とパラグラフで読みやすく

#### 3.2 ワークショップ（実践）：3つの重要セクションのコピー作成（50分）

**【ステップ1：「悩みへの共感」セクションのコピー】（15分）**
- ペルソナのPainsを具体的な「シーン」として描写
- 「そうそう、ウチも同じだ」と思わせる生々しさの作り方

**【ステップ2：「提供価値（ソリューション）」セクションのコピー】（15分）**
- 3つのKBFを、機能説明ではなく「ベネフィット」として表現
- Before/After形式での効果の可視化

**【ステップ3：CTAボタンとマイクロコピーの作成】（20分）**
- 「お問い合わせ」ではない、行動を促すCTA文言
- フォーム周辺の「不安を解消する」マイクロコピー
- 緊急性と限定性の適切な使い方

### セクション4：【実践3】コピーのブラッシュアップと編集
**時間：** 6:30:00 - 7:00:00（30分）

**内容構成：**

#### 4.1 コンセプト：「編集」がAIディレクターの最終価値（10分）

**【ステップ1：A/Bテスト用の複数案を生成】（7分）**

**【ステップ2：全体の整合性チェック】（7分）**

**【ステップ3：最終校正と出力】（6分）**
